1. Скачать проект с гитхаба (будут только добавленные и отредактированные файлы)
    https: https://github.com/StanislavO777/organizations.git
    ssh: git@github.com:StanislavO777/organizations.git
2. Запустить через php-строку битрикса скрипт из корня create_infoblock.txt (он создаст тип, структуру и сам ИБ Организации)
3. Инсталлировать композер. В корне в папке app/classes лежит класс Organizations.php (для автолоада)
4. Зайти на главную страницу index.php, скрипт автоматически через api подтянет организации, запишет их элементами ИБ
после чего выведет сообщение со статусом, автоматически перезагрузит страницу и выведет карту с организациями.
